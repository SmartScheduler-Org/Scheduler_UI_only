Recommended Setup Flow

Follow this sequence before you generate. It reduces clashes, improves slot quality, and makes later edits much easier.

1

Add Teachers

Enter instructors first so courses can be mapped to real faculty availability and workload rules.

2

Add Rooms

Create lecture and lab spaces with the right categories to avoid invalid room allocations later.

3

Add Timings

Set the meeting-time grid carefully because the generator depends directly on these slots.

4

Add Courses

Define subjects with correct room requirements and faculty mappings so the engine sees the true constraints.

5

Add Departments

Departments organize rooms, sections, and course ownership, which keeps scheduling data coherent.

6

Add Sections

Attach the right courses to the right sections before generation so required lecture and lab load is accurate.

7

Generate

Run the generator only after the base academic structure is complete and constraint data looks clean.

8

Review & Save

Inspect generated output, apply edits if needed, and save the final version once the timetable is acceptable.
